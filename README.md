Calculadora-IMC

This repository contains the Calculadora-IMC project, a web-based Body Mass Index (BMI) calculator developed using React as part of front-end training.

Live version:
https://lucashenrique1908.github.io/Calculadora-imc/

Description

Calculadora-IMC is a single-page application that allows users to calculate their Body Mass Index (BMI) by entering height and weight values. The application processes the input data in real time and displays the result in a clear and user-friendly interface.

This project was created mainly for learning and practice purposes, focusing on component-based architecture and state management.

Technologies Used

 * React

 * JavaScript

 * HTML

 * CSS

Purpose

The main goals of this project are to:

 - Practice React fundamentals and component structure

 - Work with state, events, and user input

 - Improve front-end logic and UI responsiveness

The project serves as a practical exercise in building interactive applications with React.

How to Run Locally: 

Clone the repository: git clone https://github.com/lucashenrique1908/Calculadora-imc.git

Notes

This project is part of an ongoing learning process and demonstrates the application of React concepts in a small, focused project.
